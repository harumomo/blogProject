# blogProject简介
 一个完整的博客项目，有登录注册，写博客，查看博客并评论，用户管理，博客管理功能；
 普通用户登录可以查看别人的博客，并对别人的博客评论建议，或者自己写博客；
 管理员登录还可以对用户和博客进行管理

# 环境

 Django 4.2.3   /
 Django-bootstrap3   /
